# 📊 TrackWise – Smart Portfolio Tracker

TrackWise is a modern, full-stack portfolio tracking application that helps users monitor, analyze, and manage their investments in real time. It supports multiple asset classes, live price fetching, advanced analytics, and downloadable reports — all wrapped in a clean, dark/light themed UI.

---

## 🚀 Features

### 🔹 Asset Management
- Add and manage multiple asset types:
  - 📈 **Stocks**
  - 🪙 **Cryptocurrencies**
  - 🪙 **Commodities (Gold, etc.)**
  - 🏠 **Property**
- Real-time price fetching using live APIs
- Automatic calculation of:
  - Invested amount
  - Current value
  - Profit / Loss (%)
- Asset-wise categorization and tracking

---

### 📊 Dashboard
- Total portfolio value
- Today’s profit
- Liquidity overview
- Holdings summary
- Portfolio performance visualization
- Quick insights (top gainer & loser)

---

### 📑 Reports & Insights
- Detailed portfolio analytics
- Best and worst performing assets
- Total return percentage
- Asset allocation (Pie Chart)
- Profit/Loss trend (Bar Chart)
- Time range filters:
  - Daily
  - Weekly
  - Monthly
  - Yearly
- Export reports as:
  - 📄 **PDF**
  - 📊 **CSV**

---

### 📈 Analytics Page
- Portfolio distribution visualization
- Portfolio growth trends
- Performance comparison across assets
- Clean and interactive charts powered by Chart.js

---

### ⚙️ Settings
- Dark & Light theme toggle
- Multi-currency support
- User profile settings
- Danger zone to reset portfolio

---

### 🔐 Authentication UI
- Login & Create Account screens
- Modern glass-morphism UI
- Secure layout structure (ready for backend auth integration)

---

## 🛠️ Tech Stack

### Frontend
- **React + TypeScript**
- **Chart.js** & **react-chartjs-2**
- **Lucide Icons**
- **CSS (Glassmorphism UI)**

### Backend
- **Node.js**
- **Express.js**
- **MongoDB**
- REST APIs for assets and pricing
